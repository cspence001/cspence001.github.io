Latitude and Weather Elements

The purpose of this project was to analyze Weather Data to track changes in weather when factoring in distance to the equator. Four factors: temperature, humidity, cloudiness, and wind speed were analyzed in reference towards latitude measurements of each location.

Four visualizations were rendered in the form of scatterplots to showcase these relationships. The equator position in each plot is established as the 0 degree point on the latitude axis.

This project expands on an analysis created by a python script visualizing the weather of 500+ cities across the world of varying distance from the equator. A configuration of the OpenWeatherMap API was utilized for data wrangling to consolidate the data into a dataframe displayed on the Data page.
